This is legendary pong game in python.

  <img src="Images/pong.ico" alt="pong"/>

## Functionality:

- [x] Paddle movement
- [x] Ball movement
- [x] Ball Physics
- [x] Score for each player
- [x] Ball Speed

## How to play

1. Use WA to move left peddle and Up down Arrow to move right paddle.
2. hit the ball
3. Don't hit miss and score point
4. Have fun!
